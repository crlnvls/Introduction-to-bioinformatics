# Biomolecules

## Nucleic acids

- One of the main 4 ain type of biomolecules 
- Genetic information storage 
- DNA and RNA 

## Nucleotides

- Small molecules that make up nucleic acids 
- It has 3 parts:

1. Phosphoric acid residue
2. Five-carbon cyclical sugar
3. Nitrogenous base
